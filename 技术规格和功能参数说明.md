一、产品基本信息
产品名称:颜值立方教学管理平台
系统架构:B/S 架构（Browser/Server），SaaS 模式
开发语言:前端：vue框架 / 后端：Python 3.10+
核心技术:Vue 3 + FastAPI + MySQL
接口协议：RESTful API 标准接口
实时通信：WebSocket
身份认证：OAuth2.0 + JWT (JSON Web Token) 无状态认证
加密算法：密码哈希存储, 传输路由加密
数据校验：Pydantic V2 严格数据模型验证

二、功能模块详细说明
1 教师端
工作台 (Dashboard)
（1）数据概览：实时统计学生总数（去重）、执教班级数、待批改作业数。
（2）班级卡片流：展示管理的班级，支持显示绑定的教材名称。
（3）新建班级：支持设置班级名称、封面、多选绑定课程、设定开结课时间。
班级与学员管理
（1）侧边抽屉模式：点击班级直接右侧滑出学员名单，不打断当前工作流。
（2）学员管理：支持手动添加学员、Excel 批量导入（包含数据清洗与错误报告）、移除学员。
课程资源库
（1）SaaS 授权模式：展示平台所有课程，根据权限显示“已授权”或“锁定（变暗）”状态。
（2）资源预览：支持查看课程大纲，在线预览 PDF 课件，PPT 幻灯片演示模式。
（3）作业预览：查看该课程包含的标准作业要求。
作业发布管理
（1）支持查看作业模板在不同班级的发布状态。
（2）支持分班级设置截止时间，一键发布。

2 学生端
我的课程 (Learning Center)
（1）数据扁平化：将所在班级的多门课程拆解为独立卡片展示。
（2）智能进度：卡片实时显示学习进度百分比（服务端计算）。
（3）讲师信息：动态展示授课老师的头像与职称。
沉浸式学习页
（1）布局：左侧折叠目录 + 右侧全屏阅读器。
（2）PDF 阅读：支持连续滚动阅读、缩放。
（3）PPT 演示：支持全屏放映模式，支持键盘翻页。
（4）进度追踪：阅读到底部自动标记完成，左侧目录实时打钩。
（5）状态记忆：支持断点续学，下次打开自动跳转到上次阅读页码。
作业任务中心
（1）看板视图：左侧任务列表 + 右侧数据图表（完成率环形图、紧急任务雷达）。
（2）状态流转：待完成 -> 批改中 -> 已批改。
（3）作业提交：支持富文本输入及 图片上传。
（4）前置锁：未完成课程学习前，禁止提交对应作业。
